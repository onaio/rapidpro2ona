# RapidPro to Ona 

This is a script used to make JSON form submissions to Ona forms from data received from a RapidPro flow POST webhook.

## Prerequisites

- Python 2.7
- Pip

### Development

Installing dependencies:
```
$ pip install -r requirements-dev.txt
```

Run script:

```
$ cd rapidpro2ona

$ ./__init__.py
```

#### Tests

Unittests:
```
nosetests
```
